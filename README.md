## Install required ##

```
#!cmd
sudo apt-get update
sudo apt-get install nodejs
sudo apt-get install npm

```

## Clone repo ##

```
#!cmd

git@bitbucket.org:suzu_technologies/landing-page.git
cd landing-page

```

## Install dependencies ##

```
#!cmd

sudo npm install
sudo npm install --only=production

```

## Build js/css/img files ##

```
#!cmd

gulp

```
